# Installation
To install simply run 
```
conda env create -f environment.yml
```
The codebase was based on LDM repo. Where we only changed the .yaml file to train on SEM images and tune paramters
# Training and Sampling
Data used for training can be found in the splits folder. 
The model was trained using main.py. Sampling was generated by inference_fake-many.py.

## Conditional diffusion model for generating polyp images
Training:
```bash
python latent-diffusion/main.py --base latent-diffusion/configs/latent-diffusion/<.yaml> -t --gpus 0,
```
Sampling:
We can generate images based on the condition, based on:
- Existing dataset
```bash
python inference_fake-many.py
```
