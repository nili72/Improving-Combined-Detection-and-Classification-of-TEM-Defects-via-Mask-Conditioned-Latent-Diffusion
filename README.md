# Installation
To install simply run 
```
conda env create -f environment.yml
```

# Training and Sampling
Data used for training can be found in the splits folder. 
The model was trained using main.py. Sampling was generated by inference_mask.py.

## Conditional diffusion model for generating polyp images
Training:
```bash
python latent-diffusion/main.py --base latent-diffusion/configs/latent-diffusion/kvasir-ldm-vq4-.yaml -t --gpus 0,
```
Sampling:
We can generate images based on the condition, based on:
- Existing dataset
```bash
python latent-diffusion/scripts/inference_dataset.py
```
